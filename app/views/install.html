Install Lordium

1. Download

Download Lordium from here and unzip

2. Run setup file

Go to lordium directory and run below command

./install.sh

You will see setup running and installing dependencies

3. Install dependencies

User below command to install dependencies

sudo apt-get install libpq-dev python-dev

3. Run server

After installation has completed, run below commad to activate virtual environment

. .venv/bin/activate

Finally, run the server

python manage.py runserver
